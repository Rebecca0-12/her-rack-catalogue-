# Her Rack — Curated Thrift Catalogue

**🔗 [View Live Demo](https://frayed-and-found.netlify.app)**

A modern, editorial-style thrift catalogue website designed for a curated fashion resale brand. Built as a client-style project to showcase a clean and premium shopping experience for a small thrift business — from product discovery and category browsing to easy customer enquiries through WhatsApp.

## About This Project

Her Rack was designed as a complete front-end concept for a thrift fashion business, focused on creating a polished online catalogue rather than a traditional e-commerce experience.

The goal was to give a small clothing brand a simple but professional digital presence where customers can explore available pieces, view item details, and quickly contact the seller to claim items.

A key focus of the project was making the catalogue easy to maintain. Product information is stored separately using CSV data, allowing inventory updates without manually changing the website structure.

## Features

- Curated thrift catalogue with individual product cards
- Dynamic product rendering from CSV data
- Category filtering for easier browsing
- Product details including name, price, category, and availability information
- WhatsApp claim flow for customer enquiries
- Easy inventory updates without editing the HTML
- Responsive layout across desktop, tablet, and mobile
- Clean editorial-inspired design focused on the clothing and brand identity

## Tech Stack

- HTML5
- CSS3 (CSS Grid, Flexbox, custom styling)
- Vanilla JavaScript
- CSV data integration
- Google Fonts

No frameworks or build tools — a lightweight static website designed for easy deployment and maintenance.

## How It Works

Product information is stored in a CSV file and fetched by the website. The data is then dynamically displayed as catalogue items, making it easier to update products, prices, and details without changing the code manually.

This approach allows a small business owner to manage catalogue content more efficiently while keeping the website simple and lightweight.

## Notes

This project was built as a portfolio concept exploring how small thrift businesses can create a professional online catalogue without requiring a complex e-commerce platform.

Product images and content are used for demonstration purposes. A production version would use the business owner's own inventory, photography, and contact details.
